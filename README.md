# Card Saints

A simple one-page Card Saints landing page built with Next.js and Tailwind CSS.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The main page lives in `src/app/page.tsx`, with shared metadata and fonts in
`src/app/layout.tsx`.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```
